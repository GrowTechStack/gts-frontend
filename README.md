# GrowTechStack Frontend

기술 블로그 콘텐츠를 시각적으로 제공하는 웹 어플리케이션입니다.

## 기술 스택

| 분류 | 기술 |
|------|------|
| Library | React 19 |
| Language | TypeScript |
| Build Tool | Vite 7 |
| Styling | Tailwind CSS 4.0 |
| State Management | Zustand |
| Data Fetching | TanStack Query (React Query) |
| HTTP Client | Axios |

## 주요 기능

- 기술 블로그 콘텐츠 카드 뷰 제공
- 태그 및 사이트별 필터링 조회
- 콘텐츠 통합 검색
- AI 요약 결과 대시보드
- 반응형 웹 디자인

## 환경 변수

| 변수 | 설명 |
|------|------|
| `VITE_API_URL` | API 게이트웨이 주소 |

## 로컬 개발

```bash
npm install
npm run dev
```
